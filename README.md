# Turnip Boy Robs a Bank
## How to get the apks file from your Android Phone

### Install this apk file on your phone and open it 
https://github.com/Aefyr/SAI/releases/download/4.5/SAI-4.5.apk
### Click on the Backup Button on the bottom
![Screenshot 1](https://github.com/Damian2809/turnipboyrobsabank-portmaster/blob/main/tutorial/1%20Click%20Backup.PNG)
### Click on the Select Directory Button and then choose "System File Picker"
![Screenshot 2](https://github.com/Damian2809/turnipboyrobsabank-portmaster/blob/main/tutorial/2%20Click%20Select%20Directory.PNG)
![Screenshot 3](https://github.com/Damian2809/turnipboyrobsabank-portmaster/blob/main/tutorial/3.%20Click%20System%20FIle%20Picker.PNG)
### Now create a folder and name it whatever you want, but remember it because there will be your apk stored 
![Screenshot 4](https://github.com/Damian2809/turnipboyrobsabank-portmaster/blob/main/tutorial/4.%20Create%20new%20folder%20and%20name%20it%20what%20you%20want.PNG)
### Confirm it
![Screenshot 5](https://github.com/Damian2809/turnipboyrobsabank-portmaster/blob/main/tutorial/5.%20Confirm%20the%20folder%20selection.PNG)
### Choose the game you want to get the apks file from
![Screenshot 6](https://github.com/Damian2809/turnipboyrobsabank-portmaster/blob/main/tutorial/6.%20Choose%20the%20game%20you%20want%20to%20dump.PNG)
### If you clicked on the game you want to get the apks file from then click Backup
![Screenshot 7](https://github.com/Damian2809/turnipboyrobsabank-portmaster/blob/main/tutorial/7.%20Click%20Backup.PNG)
### Now make sure all Checkboxes are crossed and then click Backup
![Screenshot 8](https://github.com/Damian2809/turnipboyrobsabank-portmaster/blob/main/tutorial/8.%20Make%20sure%20all%20croses%20are%20crossed%20and%20then%20click%20backup.PNG)
### Now you have a Backup of your game
![Screenshot 9](https://github.com/Damian2809/turnipboyrobsabank-portmaster/blob/main/tutorial/9.%20Now%20you%20have%20your%20apks%20file.PNG)
